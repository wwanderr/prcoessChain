## 🎯 学习路径推荐

### 入门路径（1-2天）

1. 阅读 [01-快速开始](./01-快速开始.md)
2. 阅读 [02-核心功能说明](./02-核心功能说明.md)
3. 浏览 [项目概要设计文档](./项目概要设计文档.md)
4. 运行项目，查看日志输出

### 深入理解路径（3-5天）

1. **必读** [代码阅读指南-完整流程详解](./代码阅读指南-完整流程详解.md)
   - 从头到尾阅读，理解7个阶段
   - 对照代码，逐个方法理解
2. **强烈推荐** [核心算法详解汇总](./核心算法详解汇总.md)
   - 重点学习：全树遍历算法、虚拟父节点批量添加
   - 了解每个算法的原理和图解
3. 阅读 [IncidentProcessChain数据结构说明](./IncidentProcessChain数据结构说明.md)
4. 阅读 [断链节点与EXPLORE节点说明](./断链节点与EXPLORE节点说明.md)
5. 使用调试器跟踪一个完整请求

### 专家路径（1-2周）

1. 完成入门和深入理解路径的所有内容
2. 阅读 [晚拆分方案-架构设计与优化](./晚拆分方案-架构设计与优化.md)
3. 阅读 [图裁剪集成说明](./图裁剪集成说明.md)
4. 阅读 [性能优化方案](./性能优化方案.md)
5. 阅读所有功能模块文档
6. 参与代码重构和优化
